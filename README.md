# FactoryFlow

FactoryFlow is a manufacturing management platform that I am building to combine production tracking, machine monitoring, maintenance reporting, analytics, and AI into one application.

The goal of this project is to build something that could realistically be used in a manufacturing environment while improving my skills in backend development, databases, APIs, and full stack development.

## Project Goals

FactoryFlow will eventually allow users to

- Manage production orders
- Track machine downtime
- Report maintenance issues
- View production analytics
- Manage employees
- Use AI to identify recurring problems and generate reports

## Why I Started This Project

I spent several years working in manufacturing before earning my Computer Science degree. During that time I saw how important good data is for production, maintenance, and operators on the floor.

I wanted to build something that combines both of those experiences into one project. Instead of making another to do list or weather app, I wanted to build software that solves problems I have actually seen in manufacturing.
